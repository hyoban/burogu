## Getting Started

1. Clone this repository or use this template
2. Change the content of `siteconfig.json` to your own
3. Deploy to Vercel
4. Add `NOTION_TOKEN` and `NOTION_DATABASE_ID` to your Vercel environment variables

That's it!
